IT operations and software development, grounded in real business problems

I'm Frank Valdez, a SUNY Old Westbury Alumni, IT operations manager, and software developer in the New York City area. Day to day, I run the technology behind a two-location grocery operation: point-of-sale and back-office systems, SQL Server data workflows, publishing, dashboards, and the user support that keeps everything moving. Before that, I worked across enterprise IT — identity and access management, device provisioning and large-scale deployments, and help desk operations.

The pattern in my work is the same everywhere: when a process is slow, repetitive, or error-prone, I build software to fix it. That has produced automation scripts, longitudinal data pipelines, internal tools, and web applications, written mostly in Python and JavaScript. The full history is on my experience page, and there's more background about me and the skills I work with.
